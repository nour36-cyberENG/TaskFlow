# TaskFlow - C++ Console Task Manager ✅

## 🎯 Overview

**TaskFlow** is a lightweight, console-based task management system developed in C++. It allows users to efficiently manage their daily tasks directly from the terminal with a clean and interactive interface.

---

## ✨ Features

- **➕ Add New Task**: Create tasks with a unique ID, title, and description.
- **📋 View All Tasks**: Display the complete list of tasks with their current status.
- **🔍 Search Task**: Find a specific task instantly using its unique ID.
- **✅ Mark as Completed**: Update the status of a task to "Completed".
- **📊 View Completed Tasks**: Filter and display only completed tasks.
- **⏳ View Pending Tasks**: Filter and display tasks that are still pending.
- **📈 Project Statistics**: Get a quick summary of total, completed, and pending tasks.

---

## 🛠️ Technologies Used

- **Language**: C++ (Standard C++17/20)
- **Libraries**: Standard Template Library (STL) - `iostream`, `string`
- **Development Environment**: Compatible with Dev-C++, Visual Studio, or any GCC compiler.
